En este readme voy a hablar sobre los trabajos realizados durante la semana 9 del curso de programación básica.

El primer trabajo consiste en escribir un programa que encuentre el número más grande entre una lista de números dados por el usuario.
La lista de números son dados de en un archivo .txt

El segundo trabajo consiste en escribir un programa que calcule el promedio y desviación estándar de un conjunto de números dados en un archivo.

En ambos trabajos se utilizaron funciones que hacen manejo dinámico de la memoria, utilizamos calloc, malloc, free
